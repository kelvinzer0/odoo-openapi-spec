REST API for Odoo

Expose any Odoo model as REST API endpoints

Transform your Odoo instance into a powerful REST API. Access any business model — Partners, Products, Sales, Purchases, Inventory, Accounting, CRM, and more — through simple HTTP endpoints.

Bearer Token Authentication

Generate secure API tokens directly from the Odoo backend. No password sharing, no complex setup. Just click Generate Token in the API Console and start making requests.

Upsert Support

Create or update records in a single API call using the upsert feature. Define a unique key (like email for partners or barcode for products) and the API handles the rest — no need to check if a record exists first.

Smart Relational Fields

No more guessing IDs. Send field names directly — "categ_id": "Electronics" — and the API auto-resolves them. If the related record doesn't exist, it's created automatically.

Interactive API Console

Built-in Scalar UI documentation right inside Odoo. Browse all endpoints, see request/response examples, and test API calls directly from your browser — no third-party tools needed.

45 Business Models

Includes REST endpoints for: Account, Journal, Move, Payment, Tax, CRM (Lead, Stage, Tag), HR (Department, Employee, Job), MRP (BOM, Production, Workorder), POS (Config, Order, Session), Product (Attribute, Category, Product, Template), Project, Purchase, Partner, Sale, Stock (Location, Move, Picking, Quant, Warehouse), and UoM.

Beginner Friendly

Clear error messages guide you to the right fields. The /api/{model}/fields endpoint shows all available fields, their types, and relationships. Domain filter documentation includes practical examples for every operator.

n8n Automation Ready

Connect to n8n for powerful workflow automation. Use the Odoo n8n Community Node to automate tasks like syncing orders, creating invoices, sending reports, and more — all without writing code.
